## Running the project
The project can be viewed and run by cloning the repo to VSCode
1. Make sure you have VS Code installed on your computer
2. Inside VS Code, press Ctrl+Shift+P to open the Command Palette
3. Type: `Git: Clone`
4. Paste the URL of the repository: `https://github.com/priscilla-cassandra/portfolio-.git`
5. Select a folder on your local machine to save repository
6. When prompted, click "Open" to open the cloned repository in VS Code
